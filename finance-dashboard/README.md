# 💰 Finance Dashboard

## 📌 Overview

This is a modern finance dashboard built using React. It allows users to track income, expenses, and financial insights through an interactive UI.



## 🚀 Features

### 📊 Dashboard

* Total Balance, Income, Expense cards
* Line chart for trends
* Pie chart for category breakdown

### 💳 Transactions

* Search transactions
* Filter (Income / Expense)
* Sort by amount
* Add, Edit, Delete (Admin only)

### 🔐 Role-Based UI

* Viewer → Read-only
* Admin → Full access

### 📈 Insights

* Highest spending category
* Total transactions
* Financial summary

---

## 🛠 Tech Stack

* React
* Context API
* CSS (Custom UI)
* LocalStorage

---

## ⚙️ Setup Instructions

```bash
git clone https://github.com/YOUR_USERNAME/finance-dashboard.git
cd finance-dashboard
npm install
npm run dev
```


## 🧠 Approach

The app is built using reusable components with centralized state using Context API. The UI dynamically changes based on user role (Admin/Viewer).



* Uses mock data
* Focused on frontend architecture and UI design
